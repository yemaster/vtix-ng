# VTIX 人工测试项目清单

> 目的：覆盖核心功能路径、关键交互、数据持久化与多端适配。
> 说明：每个条目建议在"桌面端 + 移动端"均测试一遍（如适用）。

## 1. 页面的访问和正常显示

检查下列页面是否能正常访问：

首页，公告列表，公告详情，题库列表，题库广场，做题页面，练习记录，我的错题，登录页，注册页，个人空间页（设置），站内消息，题库大乱斗，管理后台。

## 2. 登录 / 注册
- 登录页输入框与按钮可输入、可点击
- 注册页输入框与按钮可输入、可点击
- 登录/注册页返回首页跳转正常

## 3. 首页（Home）
- 统计卡片显示数值/标签/进度条
- Hero 按钮跳转至题库
- 推荐题库列表可点击进入
- 移动端自适应布局正常

## 4. 题库列表（Question Bank）
- 题库列表加载成功 / 失败提示
- 搜索过滤（标题/编号/年份）
- 分类标签筛选
- 分页切换与每页数量切换
- 题库卡片点击进入测试页
- 卡片涟漪效果 + 延迟跳转正常
- 空列表状态显示正确

## 4.1 题库广场（Plaza）
- 广场列表加载成功，分页正常
- 搜索/筛选生效
- 题库反应（点赞/收藏）可点击且状态持久
- 点击卡片进入题库详情

## 5. 练习页基础（TestView）
- 标题与题目内容正确显示
- 单选/多选/判断/填空 UI 正常显示
- 选项整行可点击选中（不必点到控件）
- 提交后正确/错误/漏选样式显示
- 上一题/下一题按钮可用性正确
- 题目编号列表可点击跳转

## 6. 练习模式
- 顺序练习：题目顺序与题库一致
- 乱序练习：题目顺序随机，选项顺序随机
- 自定义练习：配置卡片 → 选择题型/是否打乱 → 开始练习
- 错题回顾：仅包含错题
- 模拟考试：按配置抽题（含类型配比）

## 7. AI 题目解析
- 做题页可触发"AI 解析"
- 解析以流式逐字展示（SSE）
- 解析结果支持 Markdown 渲染
- 未登录 / 未配置 AI Key 时给出明确提示
- 同一题重复请求复用缓存，不重复请求后端
- 个人空间"设置"页可保存自定义 AI 服务商 / API Key / 模型
- 切换服务商后"获取可用模型"能拉到模型列表

## 8. 键盘快捷键
- Enter 提交
- 左右方向键切换题目
- Q/W/E/R/T/Y 选择前 6 个选项
- 输入框内输入不误触发快捷键

## 9. 进度与统计展示
- 桌面端选题卡片顶部：正确数/总数/正确率（非考试）
- 模拟考试：显示作答数/总数/总用时
- 移动端底部菜单：练习模式名 + 正确率/总用时；选题显示题号进度
- 颜色提示醒目且不遮挡

## 10. TestView 内做题记录
- 仅在首次操作后才保存记录
- 记录列表显示：模式/时间/用时/进度
- 继续练习加载记录成功
- 删除记录成功
- 导出/导入（当前题库记录）

## 11. 全局做题记录
- 列表显示所有题库记录
- 继续 / 删除操作正确
- 进度条与百分比显示正确
- 导出全部记录 / 导入记录合并（最多保留上限条数）
- 多端增量同步：A 端更新后 B 端能拉到最新

## 12. 移动端底部菜单
- 菜单/练习/选题/记录四项切换正常
- 高亮状态正确
- 菜单页 / 记录页不被底部菜单遮挡
- 切换时不出现重叠页面

## 13. 本地存储与持久化
- 本地存储 key：`vtixSave` 读写正常
- 最多保留上限条记录（按更新时间）
- 刷新后可恢复最后一条记录
- 指定记录跳转加载：`/t/:id?record=xxx`

## 14. 题库大乱斗（Brawl）
- 大厅连接成功，显示在线用户/题库
- 大厅聊天可发送/接收
- 选择题库后开始匹配，能匹配到对手
- 对战中答题提交、回合切换正常
- 对手掉线时显示暂停提示
- 对战结束显示胜负与得分
- 用户空间页展示历史战绩

## 15. 站内消息
- 未读消息数角标正确
- 消息列表加载，已读状态自动更新
- 题库审核通过/拒绝时创建者收到消息
- 点击消息内链接跳转正确

## 16. 题库管理与审核
- 普通用户创建私有题库可提交发布申请
- 管理员待审核列表能看到申请
- 审核通过/拒绝后题库公开状态正确变更
- 管理员可导出/导入题库

## 17. 错误与边界
- 空题库/无数据时页面提示正常
- 题型缺失/异常不崩溃
- 数据量很大时滚动与性能表现

## 18. 视觉与交互一致性
- 按钮样式一致（无意外全局污染）
- 悬停/点击/禁用状态表现正常
- 卡片阴影与间距一致
