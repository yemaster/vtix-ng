# VTIX 人工测试项目清单

> 目的：覆盖核心功能路径、关键交互、数据持久化与多端适配。
> 说明：每个条目建议在“桌面端 + 移动端”均测试一遍（如适用）。

## 1. 全局导航与路由
- 首页可进入，导航栏高亮与当前路由匹配
- 导航项：首页 / 题库 / 做题记录 可跳转
- 移动端顶部导航：Logo + 菜单抽屉展开/关闭
- 抽屉内导航项可跳转并自动关闭抽屉
- TestView 移动端隐藏顶部导航（底部菜单接管）

## 2. 登录 / 注册
- 登录页输入框与按钮可输入、可点击
- 注册页输入框与按钮可输入、可点击
- 登录/注册页返回首页跳转正常

## 3. 首页（Home）
- 统计卡片显示数值/标签/进度条
- Hero 按钮跳转至题库
- 推荐题库列表可点击进入
- 移动端自适应布局正常

## 4. 题库列表（Question Bank）
- 题库列表加载成功 / 失败提示
- 搜索过滤（标题/编号/年份）
- 分类标签筛选
- 分页切换与每页数量切换
- 题库卡片点击进入测试页
- 卡片涟漪效果 + 延迟跳转正常
- 空列表状态显示正确

## 5. 练习页基础（TestView）
- 标题与题目内容正确显示
- 单选/多选/判断/填空 UI 正常显示
- 选项整行可点击选中（不必点到控件）
- 提交后正确/错误/漏选样式显示
- 上一题/下一题按钮可用性正确
- 题目编号列表可点击跳转

## 6. 练习模式
- 顺序练习：题目顺序与题库一致
- 乱序练习：题目顺序随机，选项顺序随机
- 自定义练习：配置卡片 → 选择题型/是否打乱 → 开始练习
- 错题回顾：仅包含错题
- 模拟考试：按配置抽题（含类型配比）

## 7. 键盘快捷键
- Enter 提交
- 左右方向键切换题目
- Q/W/E/R/T/Y 选择前 6 个选项
- 输入框内输入不误触发快捷键

## 8. 进度与统计展示
- 桌面端选题卡片顶部：正确数/总数/正确率（非考试）
- 模拟考试：显示作答数/总数/总用时
- 移动端底部菜单：练习模式名 + 正确率/总用时；选题显示题号进度
- 颜色提示醒目且不遮挡

## 9. TestView 内做题记录
- 仅在首次操作后才保存记录
- 记录列表显示：模式/时间/用时/进度
- 继续练习加载记录成功
- 删除记录成功
- 导出/导入（当前题库记录）

## 10. 全局做题记录
- 列表显示所有题库记录
- 继续 / 删除操作正确
- 进度条与百分比显示正确
- 导出全部记录 / 导入记录合并（最多保留 10 条）

## 11. 移动端底部菜单
- 菜单/练习/选题/记录四项切换正常
- 高亮状态正确
- 菜单页 / 记录页不被底部菜单遮挡
- 切换时不出现重叠页面

## 12. 本地存储与持久化
- 本地存储 key：`vtixSave` 读写正常
- 最多保留 10 条记录（按更新时间）
- 刷新后可恢复最后一条记录
- 指定记录跳转加载：`/t/:id?record=xxx`

## 13. 错误与边界
- 空题库/无数据时页面提示正常
- 题型缺失/异常不崩溃
- 数据量很大时滚动与性能表现

## 14. 视觉与交互一致性
- 按钮样式一致（无意外全局污染）
- 悬停/点击/禁用状态表现正常
- 卡片阴影与间距一致
