# 常量

## 题目类型(ProblemType)

### 描述

题目类型常量值

### 值

| 常量名        | 值  | 描述         |
|---------------|-----|--------------|
| NULL_PROBLEM  | 0   | 空题目类型   |
| SINGLE_CHOOSE | 1   | 单选题       |
| MULTI_CHOOSE  | 2   | 多选题       |
| FILL          | 3   | 填空题       |
| JUDGEMENT     | 4   | 判断题       |
